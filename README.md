## 目录结构  
```
├── public                          // 静态页面 
├── src                             // 主目录
    ├── assets                      // 静态资源
    ├── components                  // 通用组件
    ├── directives                  // 通用指令
    ├── plugins                     // 通用插件
    ├── router                      // 路由
    ├── store                       // ts版 vuex 配置
    ├── styles                      // 全局样式
    ├── types                       // 全局注入
    ├── views                       // 页面视图
    ├── App.vue                     // 页面主入口
    ├── main.ts                     // 脚本主入口
    ├── shims-tsx.d.ts              // ts补充声明
    ├── shims-vue.d.ts              // VUE自定义声明
├── package.json                    // 依赖 
├── README.md                       // 项目 readme
├── tsconfig.json                   // ts 配置
├── tslint.json                     // tslint 配置
└── vue.config.js                   // 扩展webpack 配置  
```
